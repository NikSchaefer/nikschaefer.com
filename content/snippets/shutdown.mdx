tag: "wifi"

## Shut down all computers on a network on windows

```bash
shutdown -i
```

## Shut down all computers on a network on linux

```bash
nmap -sP
```

# Shut down computer with IP address on the same WiFi

```bash
shutdown -i
```

Then click add and type in the IP address of the computer you want to shut down.
